¿Sobre que trata el ramo?

En el ramo de Fundamentos de Programacion se usa Python y se empieza desde un nivel basico. En esta pagina se podran encontrar links a distintos lugares donde pueden aprender por su cuenta, ver que profesores se recomiendan para este, y evaluaciones ya realizadas para que puedan ejercitar este ramo y ver como son las PEP

¿Que temas en especificos se veran?

Se empiezan viendo strings con el tipico ejemplo de "Hola Mundo", los diferentes tipos de objetos que hay (string,int,float) y los usos de simbolos matematicos (*,//,&,**). Despues se veran las listas y funciones que los objetos pueden usar (.islower(),.sum()) Finalmente se veran los ciclos iterativos (while), en los cuales se itera usualmente desde un 0 hasta un cierto numero, y al final de este while se usa un i++ para aumentar el contandor en uno para asi terminar cuando se haya indicado
